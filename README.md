**iPhone Home Screen UI**
A simple front-end recreation of two iPhone home screens featuring a collection of app icons arranged in a clean and familiar iOS-style layout.
Overview
This project focuses on recreating the appearance of two iPhone home screens using modern front-end technologies. The goal is to practice layout design, responsive positioning, and attention to visual detail while maintaining a clean and organized codebase.
Features
Two iPhone home screen layouts
Realistic app icon arrangement
Responsive design
Clean and reusable components
Simple and easy-to-read code structure

**Technologies Used**
HTML5
CSS3
